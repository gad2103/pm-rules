A working project

AngularJS and ExpressJS with Google Authentication project to start new projects.

Features

  • Google Authentication
  • MongoDB storage

Authentication Flow

  • By default accounts are auto approved.
  • The system settings can disable the auto approval
  • First account can change everything (admin role)
